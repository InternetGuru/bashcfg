### How do I get set up? ###

* Add the following lines into your `.bashrc` file

```
#!shell

echo "IG bashcfg, version $(< $HOME/"bashcfg/VERSION")"
source "$HOME/bashcfg/common"
source "$HOME/bashcfg/git_functions"
PS1="$MYPS1"

```

* Configuration: see used variables
* Dependencies: none
